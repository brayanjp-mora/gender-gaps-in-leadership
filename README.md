# Gender Gaps in Leadership: A Cross-Country Analysis

⸻

## Overview

This project analyzes gender disparities in leadership using the Sustainable Development Goal (SDG) indicator:
- SDG 5.5.2: Proportion of women in managerial positions
The analysis compares trends across Colombia and South Africa using data from the World Bank.

⸻

## Objective

To explore how women’s representation in managerial roles has evolved over time and identify differences across countries.

⸻

## Data
	•	Source: World Bank Open Data
	•	Indicator: Proportion of women in managerial positions (%)
	•	Time period: [e.g., 2000–2023]
